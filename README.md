# swiftra-example

An example of [Swiftra](https://github.com/takebayashi/swiftra).

## Usage

```
% git clone https://github.com/takebayashi/swiftra-example.git
% cd swiftra-example
% swift build
% ./.build/debug/swiftra-example
```
